# База пользователей

Проект реализован в рамках тестового задания на позицию стажера React Frontend-разработчика.

## Стек технологий

- TypeScript
- Webpack + Npm
- React 16
- Ant Design 5 + styled-components
- React Router 6
- TanStack Query 4 + axios
- dayjs
- Архитектура FSD (Feature-Sliced Design)

## Функционал

- Страница авторизации (логин/пароль: admin/admin)
- Страница списка пользователей с данными из mockapi.io
- Создание, редактирование и удаление пользователей через модальные окна
- Валидация форм
- Защита маршрутов (проверка авторизации)
- Страница 404

## Запуск

```bash
npm install
npm run start
```

Приложение запустится на http://localhost:3000

## Структура проекта

```
src/
├── app/           -- точка входа, провайдеры, роутинг, стили
├── pages/         -- страницы (Login, Users, 404)
├── features/      -- фичи (auth, users)
└── shared/        -- общие утилиты, api-клиент, конфиг
```
